# News Management System

A modern News Management System built with Laravel, featuring role-based access control, multi-guard authentication, content management, SEO optimization, and third-party service integrations.

## Features

- Multi-Guard Authentication
- Role-Based Access Control (RBAC)
- News & Category Management (CRUD)
- Comments & Nested Replies
- Tags Management
- Newsletter Subscription
- Contact Messages Dashboard
- Dynamic Site Settings
- SEO Optimization
- Third-Party API Integration
- Email Verification & Password Reset
- User Profile Management

## Technologies

- Laravel
- PHP
- MySQL
- Bootstrap
- JavaScript
- Microsoft Translator API
- Spatie Laravel Permission

## Installation

```bash
git clone https://github.com/omar-mohamedomar/news.git

cd news

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
